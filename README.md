## Benefícios dessa disciplina
A matéria de laboratorio programação é o coração da formação de um desenvolvedor aqui os beneficios vão alem da escrita do código:
1. **Resolução de problemas complexos:** Você aprendera a decompor grandes desafiios em pequenas tarefas tratáveis.
2. **"Mentalidade mão na massa:** A disciplina foca na experimentação. O erro aqui não é um fracasso, mas um passo nescessário para o aprendizado.
